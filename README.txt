# Collapsible Sidebar

A simple collapsible sidebar built using **HTML**, **CSS**, and **JavaScript**.

## Features
- Slide-in/slide-out animation for sidebar.
- Responsive design for mobile and desktop.
- Toggle button with icon change (hamburger ↔ close icon).
- Smooth transition effects.
- Font Awesome icons for navigation items.

## Files
- index.html — Structure of the sidebar and content.
- style.css — Styling for sidebar, toggle button, and responsiveness.
- main.js — Handles sidebar toggle functionality and icon switching.

## How to Use
1. Open index.html in your browser.
2. Click the menu button to open or close the sidebar.
3. Works on both desktop and mobile.

## Customization
- Change the sidebar background color in style.css:
.sidebar {
  background-color: #2c3e50;
}

- Modify icons or menu items in the <ul> inside index.html.
